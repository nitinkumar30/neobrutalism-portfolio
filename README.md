## 🚀 Getting Started

### ✅ Prerequisites

- **Node.js** 18.x+ (20.x recommended)
- **npm** 9.x+
- A terminal (PowerShell, bash, zsh)

### 📥 Installation

```bash
# 1️⃣ Clone the repository
git clone https://github.com/nitinkumar30/neobrutalism-portfolio.git

# 2️⃣ Navigate into the project
cd neobrutalism-portfolio

# 3️⃣ Install dependencies (use --legacy-peer-deps for React 19 compat)
npm install --legacy-peer-deps
```

> ⚠️ `--legacy-peer-deps` is needed because `@icons-pack/react-simple-icons` hasn't declared React 19 peer support yet.

### 💻 Development Server

```bash
npm run dev
```

Opens at 👉 **http://localhost:3000** 🚀

- Hot-reload on every file change
- No turbopack (uses webpack for maximum compatibility)

### 🏗️ Production Build

```bash
npm run build
```

- Compiles all 14 pages as static HTML
- Output in `.next/` directory
- Lint & type-checking run automatically

### ▶️ Start Production Server

```bash
npm start
```

Serves the built static site on **http://localhost:3000**

### ✅ Lint

```bash
npm run lint
```

Runs ESLint with Next.js config.

## 🎨 Design Tokens

Defined in `globals.css` using Tailwind v4 `@theme inline`:

| Token | Light | Dark |
|-------|-------|------|
| `--background` | `oklch(95.38% 0.0357 72.89)` | `oklch(26.86% 0.0327 60.06)` |
| `--main` | `oklch(72.27% 0.1894 50.19)` | `oklch(67.56% 0.1796 49.61)` |
| `--foreground` | `oklch(0% 0 0)` | `oklch(92.49% 0 0)` |
| `--shadow` | `4px 4px 0px 0px var(--border)` | Same |
